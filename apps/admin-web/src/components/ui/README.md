# shadcn/ui components

Install components here with the shadcn CLI, e.g.:

```bash
cd apps/admin-web
npx shadcn@latest add button card table badge dialog input select dropdown-menu
```

`components.json` is already configured (new-york style, neutral base color, CSS
variables, Lucide icons). Generated files land in this folder and are imported as
`@/components/ui/<component>`.
